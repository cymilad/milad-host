### Run Localhost

To run **milad-host** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/cymilad/milad-host.git
```

Windows:

```bash
git clone https://github.com/cymilad/milad-host.git
```
